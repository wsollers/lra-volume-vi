\subsection*{Partial Derivatives}

\begin{definition*}[Partial Derivative]
For a function \(f(x,y)\), the partial derivative \(f_x\) differentiates with
respect to \(x\) while holding \(y\) constant. The derivative \(f_y\) does the
same with respect to \(y\).
\end{definition*}

\begin{remark*}[Several-Variable Chain Rule]
The chain rule in several variables tracks every path by which a variable can
affect the output. In computations, draw the dependency diagram before
writing the derivative formula.
\end{remark*}

\begin{definition*}[Tangent Plane]
For \(z=f(x,y)\), the tangent plane at \((a,b)\) is
\[
  z=f(a,b)+f_x(a,b)(x-a)+f_y(a,b)(y-b),
\]
when the required differentiability hypotheses hold.
\end{definition*}
