\subsection*{Sequences}

\begin{definition*}[Sequence]
A sequence is a function from positive integers to numbers, usually written
\((a_n)\).
\end{definition*}

\begin{definition*}[Convergence of a Sequence]
The sequence \((a_n)\) converges to \(L\) when \(a_n\) approaches \(L\) as
\(n\to\infty\). Computationally, this is evaluated using algebraic
simplification, standard limits, squeezing, monotonicity, and comparison.
\end{definition*}
