\subsection*{Structures as Mathematical Objects}
\label{subsec:structures-as-mathematical-objects}

\begin{remark*}[Exposition]
An algebraic structure is not only a set.  It is a carrier set together with
specified operations, relations, distinguished elements, and laws.  The
signature records which symbols are available; a model interprets those symbols
on a concrete carrier.
\end{remark*}
