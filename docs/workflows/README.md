# LRA Workflows

This directory holds operational workflows that are too procedural for the
core governance standards.

Modules:

- `knowledge-extraction.md`
- `volume-cleanup.md`
- `generated-wrapper-sync.md`
- `decoration-audit.md`

Use these files when a task asks how to run or review a process. Use
`docs/governance/` for the rules the process enforces and `docs/architecture/`
for ownership and system boundaries.
