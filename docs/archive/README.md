# LRA Governance Archive

This directory is reserved for retired design notes, obsolete assumptions, and
historical material that should not remain active policy.

Active governance belongs under:

- `docs/governance/`
- `docs/architecture/`
- `docs/governance/repo-overlays/`
- `docs/workflows/`

Do not place current rules here. If an old assumption is retained for context,
make clear that it is archival and not an active instruction.
